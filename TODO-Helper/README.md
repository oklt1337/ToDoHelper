# TODO-Helper

## File format

Comments must conform the following format:

```c#
// TODO
// # Title Title of task
// # Assignee My Name
// # DueDate MM/dd/yyyy HH:mm
// # Priority Minor
// # Description Your description...
// which can also be multiline as long as it's the last item
```

or

```c#
/*
 * TODO
 * # Title Title of task
 * # Assignee My Name
 * # DueDate MM/dd/yyyy HH:mm
 * # Priority Minor
 * # Description Your description...
 * which can also be multiline as long as it's the last item
 */
```

Priority's:

Trivial, Minor, Major, Critical, Blocker
